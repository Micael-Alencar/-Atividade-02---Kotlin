# Atividade 02 - Kotlin

Projeto desenvolvido em Kotlin com foco na prática de lógica de programação e resolução de exercícios no console.

## Sobre o projeto
Esta atividade reúne três exercícios práticos que trabalham conceitos fundamentais da programação, como entrada de dados, estruturas de repetição, condicionais, contadores e cálculos.

## Exercícios desenvolvidos
- Identificação do maior e menor número digitado, encerrando a execução ao informar 0
- Análise de dados de cidades de São Paulo, incluindo média de acidentes, média de veículos e cidade com maior índice de acidentes
- Contagem de números digitados que estão entre 100 e 200, finalizando com 0

## Objetivo
Praticar os conceitos básicos da linguagem Kotlin e desenvolver lógica de programação por meio de exercícios práticos.

## Tecnologias utilizadas
- Kotlin
- IntelliJ IDEA
